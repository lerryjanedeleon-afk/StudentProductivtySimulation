===============================================
STUDENT PRODUCTIVITY BEHAVIOR SIMULATION SYSTEM
===============================================

PROJECT DESCRIPTION
-----------------------------------------------

The Student Productivity Behavior Simulation System
is a web-based simulation project that analyzes
how student activities affect productivity and
academic performance.

The system allows users to enter their daily or
weekly activities and predicts productivity levels,
academic performance, and provides personalized
time management recommendations.

This project is based on Social Science Modeling
and Simulation, specifically Behavior Modeling.

===============================================
PROBLEM STATEMENT
===============================================

Many students struggle with time management due
to activities such as excessive social media use,
gaming, insufficient sleep, and lack of study time.

This system helps students understand how these
activities may affect their productivity and
academic performance.

===============================================
HOW TO RUN THE PROJECT
===============================================

1. Open Visual Studio Code.
2. Open the project folder.
3. Open index.html.
4. Run using:
   - Live Server, or
   - Double-click index.html

5. Enter:
   - First Name
   - Last Name
   - Social Media Hours
   - Study Hours
   - Sleep Hours
   - Gaming Hours
   - Time Period (Daily or Weekly)

6. Click "Run Simulation".

7. View the generated:
   - Productivity Score
   - Productivity Level
   - Academic Performance Prediction
   - Time Management Advice
